## 买飞机票

机票价格按照淡季旺季、头等舱和经济舱收费、输入机票原价、月份和头等舱或经济舱。按照如下规则计算机票价格:旺季(5-10月)头等舱9折，经济舱8.5折，淡季(11月到来年4月)头等舱7折，经济舱6.5折。

```java
public class Test1 {
    public static void main(String[] args) {

        //1.键盘录入 机票原价、月份和头等舱或经济舱
        Scanner sc = new Scanner(System.in);
        System.out.println("请输入机票的原价");
        int ticket = sc.nextInt();
        System.out.println("请输入当前的月份");
        int month = sc.nextInt();
        System.out.println("请输入购买的舱位 0 头等舱  1 经济舱");
        int seat = sc.nextInt();
        //2.先判断是淡季还是旺季
        //3.继续判断当前机票是经济舱还是头等舱
        //4.根据实际情况计算出实际的价格
        

        if (month >= 5 && month <= 10) {
            // 旺季
            ticket = getPrice(ticket, seat, 0.9, 0.85);
        } else if ((month <= 4 && month >= 1) || (month >= 11 && month <= 12)) {
            //淡季
            ticket = getPrice(ticket, seat, 0.7, 0.6);
        }
        System.out.println(ticket);
    }

    public static int getPrice(int ticket, int seat, double v0, double v1) {
        if (seat == 0) {
            //头等舱
            ticket = (int) (ticket * v0);
        } else if (seat == 1) {
            //经济舱
            ticket = (int) (ticket * v1);
        } else {
            System.out.println("没有这个舱位");
        }
        return ticket;
    }
}
```

## 求质数

判断 101-200 之间有多少个素数，并输出所有素数。

```java
public class Test2 {
    public static void main(String[] args) {
        int start = 101;
        int end = 200;

        int count = 0;//计数器
        for (int i = start; i <= end; i++) {
            boolean flag = true;
            for (int j = 2; j < i; j++) {
                if (i % j == 0) {
                    flag = false;
                    break;
                }
            }
            if (flag) {
                System.out.println("当前数字" + i + "是质数");
                count++;
            }
        }
        System.out.println(start + "~" + end + "之间有" + count + "个质数");
    }
}
```

## 开发验证码

定义方法实现随机产生一个5位的验证码

验证码格式：长度为5 前四位大写字母或者小写字母  最后一位是数字

```java
import java.util.Random;

public class Test3 {
    public static void main(String[] args) {
        // 生成 a-z A-Z 的数组
        char[] chs = getChars();

        //2.生成4次字母随机数，在成一个数字随机数
        String str = "";
        str = getStr(chs, str);
        System.out.println(str);

    }

    private static String getStr(char[] chs, String str) {
        Random r = new Random();
        for (int i = 0; i < 4; i++) {
                int randomIndex = r.nextInt(chs.length);
                str += chs[randomIndex];
        }
        int sum = r.nextInt(10);
        str += sum;
        return str;
    }

    private static char[] getChars() {
        //1.定义一个数组存储大写字母和小写字母
        char[] chs = new char[52];
        for (int i = 0; i < chs.length; i++) {
            // ASCII码表
            if (i <= 25) {
                chs[i] = (char) (97 + i);

            } else {
                chs[i] = (char) (65 + i - 26);
            }
        }
        return chs;
    }
}
```

            newArr[i] = arr[i];

## 评委打分

在唱歌比赛中，有6名评委给选手打分，分数范围是[0-100]之间的整数。选手的最后得分为:去掉最
高分、最低分后的4个评委的平均分，请完成上述过程并计算出选手的得分。

```java
import java.util.Scanner;

public class Test5 {
    public static void main(String[] args) {
        int[] arr = new int[6];
        Scanner sc = new Scanner(System.in);

        //键盘录入 5个数
        for (int i = 0; i < arr.length; ) {
            System.out.println("输入评分");
            int score = sc.nextInt();
            if (score >= 0 && score <= 100) {
                arr[i] = score;
                i++;
            } else {
                System.out.println("成绩超出范围，继续录入，当前i为：" + i);
            }

        }

        //获取数组的最大值
        int max = getMax(arr);
        //获取数组的最小值
        int min = getMin(arr);

        //求出 (总和 - 最大值 - 最小值) / 4
        int result = getResult(arr, max, min);
        //输出结果
        System.out.println("选手得分：" + result);
    }

    private static int getResult(int[] arr, int max, int small) {
        int sum = 0;
        for (int i = 0; i < arr.length; i++) {
            sum += arr[i];
        }
        int result = (sum - max - small) / arr.length - 2;
        return result;
    }

    private static int getMin(int[] arr) {
        int small = arr[0];
        for (int i = 1; i < arr.length; i++) {
            if (arr[i] < small) {
                small = arr[i];
            }
        }
        return small;
    }

    private static int getMax(int[] arr) {
        int max = arr[0];
        for (int i = 1; i < arr.length; i++) {
            if (arr[i] > max) {
                max = arr[i];
            }
        }
        return max;
    }
}
```

## 数字加密

某系统的数字密码(大于0)，比如1983，采用加密方式进行传输。
规则如下:
先得到每位数，然后每位数都加上5，再对10求余，最后将所有数字反转，得到一串新数。

```java
public class Test6 {
    public static void main(String[] args) {
        int[] arr = {1, 9, 8, 3};

        //每位数都加上5
        for (int i = 0; i < arr.length; i++) {
            arr[i] += 5;
        }
        //再对10求余
        for (int i = 0; i < arr.length; i++) {
            arr[i] %= 10;
        }
        //翻转数组
        for (int i = 0, j = arr.length - 1; i < j; i++, j--) {
            int temp = arr[i];
            arr[i] = arr[j];
            arr[j] = temp;

        }
        int number = 0;
        for (int i = 0; i < arr.length; i++) {
            number = number * 10 + arr[i];
            System.out.println(number);
        }
    }
}
```

## 数字解密

```java
public class Test8 {
    public static void main(String[] args) {
        //先得到每位数，然后每位数都加上5，再对10求余，最后将所有数字反转，得到一串新数
        int[] arr = {8, 3, 4, 6};

        // 2.翻转数组
        for (int i = 0, j = arr.length - 1; i < j; i++, j--) {
            int temp = arr[i];
            arr[i] = arr[j];
            arr[j] = temp;
        }

        // 3.由于加密是通过对10取余的方式获取的
        // 所以在解密的时候需要判断，0~4之间+10，5~9之间数字不变
        for (int i = 0; i < arr.length; i++) {
            if (arr[i] >= 0 && arr[i] <= 4) {
                arr[i] += 10;
            }
        }
        //4.每一位都减5
        for (int i = 0; i < arr.length; i++) {
            arr[i] -= 5;
        }
        //5.将数组里面的的每一位数字拼接成最终结果
        int number = 0;
        for (int i = 0; i < arr.length; i++) {
            number = number * 10 + arr[i];
        }

        //打印最终结果
        System.out.println(number);//1983
    }
}
```
