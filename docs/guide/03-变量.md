## 变量的使用场景

当某个数据经常发生改变的时候，我们也可以用变量存储。当数据变化时，只要修改变量里面的记录即可。

## 变量的定义格式

**数据类型  变量名 = 数据值;**

- 数据类型：为空间中存储的数据，加入类型【限制】
- 变量名：为空间（小盒子）取名
- 数据值：存在空间里的数值
- 等号：将右边的值赋值给左边的变量

| 数据类型 | 关键字 |
| -------- | ------ |
| int      | 整型   |
| double   | 浮点数 |

## 变量的使用方式

1. 输出打印

   ```java
   int num = 6;
   System.out.println(num);

2. 参与计算

   ```java
   int num = 6;
   int b = 4
   System.out.println(num + b);
   ```

3. 修改记录的值

   ```java 
   int a = 1;
   int b = 10;
   System.out.println(a);
   a = b;
   System.out.println(a);
   ```
