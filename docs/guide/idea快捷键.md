## 快捷键
| 组合键                         | 作用                          |
| ------------------------------ | ----------------------------- |
| `Ctrl` + `Alt` + `M`           | 快速抽取方法                  |
| `Ctrl` + `Alt` + `T`           | 选中代码快捷自动生成try catch |
| `Shift` + `Alt` + `上下方向键` | 快捷移动光标所在行            |
| `Ctrl` + `D`                   | 快捷复制粘贴                  |
| `Ctrl` + `Alt` + `V`           | 给方法返回值快速生成变量      |
| `Ctrl` + `W`                   | 选词                          |
| `Alt` + `Enter`                | 查看idea给的建议              |
| `Alt` + `Insert`               | 快速添加构造方法              |

## 插件推荐
